# 楽天市場 商品情報スクレイパー

楽天市場の検索結果から商品情報を一括取得し、Excel ファイルに出力するデスクトップツールです。  
キーワードとページ数を指定するだけで、GUI 操作のみで使用できます。

## 取得できる項目

| 項目 | 説明 |
|------|------|
| 商品名 | 商品のフルタイトル |
| 価格（円） | 最低価格（税込） |
| 価格幅 | SKU複数の場合の価格範囲（例: 1000/5000） |
| 評価 | レビュースコア（0〜5） |
| レビュー数 | 累計レビュー件数 |
| ショップ名 | 出店ショップ名 |
| 商品URL | 楽天市場の商品ページURL |
