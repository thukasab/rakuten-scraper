Install

インストール方法

1

リポジトリをクローン

# リポジトリを取得
git clone https://github.com/thukasab/rakuten-scraper.git
cd rakuten-scraper
2

依存ライブラリをインストール

pip install requests beautifulsoup4 openpyxl
3

GUI を起動

python gui_app.py
4

CLI として使う場合

# キーワード「コーヒー」を 5 ページ取得
python scraper.py コーヒー --pages 5 --output result.xlsx
